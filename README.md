# TopBuilder

TopBuilder is a bilingual contractor marketplace prototype for homeowners and renovation professionals.

Live site: https://alex-xi-chen.github.io/TopBuilder/

## Features

- English and Chinese language toggle
- Contractor search and filtering
- Contractor profile pages
- Homeowner login/register mock flow
- Saved contractor/account area
- Contractor onboarding form
- Professional project imagery and Top Builder branding
- Responsive desktop and mobile layout

## Local Development

```bash
npm run dev
```

Then open:

```text
http://localhost:4173
```

## Deployment

This app is published with GitHub Pages from the `gh-pages` branch.
